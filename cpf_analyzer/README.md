# Analisador Inteligente de CPF (simulado) — Python

Projeto de portfólio para demonstrar:
- Normalização de dados (strip de caracteres)
- Validação de CPF (dígitos verificadores)
- Heurísticas de qualidade / padrões suspeitos (simuladas)
- Geração de CPFs fictícios válidos para testes
- CLI + relatório em JSON
- Testes automatizados (pytest)

> ⚠️ **Aviso**: Este projeto é **simulado** e educacional.  
> Não use para verificar identidade real ou tomar decisões legais/financeiras.
